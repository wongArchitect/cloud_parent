项目需要使用Redis来做缓存，研究了一下如何将其与Spring Boot整合。网上的demo要么就是太过于庞大，要么就是版本过于陈旧，配置时候会有各种坑。因此自己在踩过了各种坑之后，写一个小demo来记录下：

1.项目结构：

2.pom的依赖配置：

本人使用的Spring Boot是2.0.4.RELEASE版本的：

  1.     <parent>
  2.         <groupId>org.springframework.boot</groupId>
  3.         <artifactId>spring-boot-starter-parent</artifactId>
  4.         <version>2.0.4.RELEASE</version>
  5.         <relativePath/> <!-- lookup parent from repository -->
  6.     </parent>

添加redis的依赖：

 

  1.         <dependency>
  2.             <groupId>org.springframework.boot</groupId>
  3.             <artifactId>spring-boot-starter-data-redis</artifactId>
  4.         </dependency>

 

添加common-pool2的依赖：

  1.         <dependency>
  2.             <groupId>org.apache.commons</groupId>
  3.             <artifactId>commons-pool2</artifactId>
  4.             <version>2.0</version>
  5.         </dependency>

 

我在这里尝试过不加version，让springboot自己选择最合适的版本，结果会报错。参考别人的例子，选择2.0版本，没有问题。

3.修改application.yml：

这里的host填写自己redis的IP，timeout别设置为0，否则运行会报错。

redis 单机配置

  1. spring:
  2.   redis:
  3.     host: 
  4.     port: 6379
  5.     lettuce:
  6.       pool:
  7.         max-wait: 100000
  8.         max-idle: 10
  9.         max-active: 100
  10.     timeout: 5000

 

redis 集群配置

  1.   redis:
  2.     database: 0
  3.     # 集群设置 begin
  4.     cluster:
  5.       nodes:
  6.         - 10.217.17.70:7000
  7.         - 10.217.17.74:7000
  8.         - 10.217.17.75:7000
  9.       max-redirects: 3 # 获取失败 最大重定向次数
  10.     #集群设置 end
  11.     #单节点 begin
  12. #    host: 10.217.17.74
  13. #    port: 7000
  14.     #单节点 end
  15.     lettuce:
  16.       pool:
  17.         max-wait: 100000
  18.         max-idle: 10
  19.         max-active: 100
  20.     timeout: 5000

 

 

3.编写dao层 ：

  1. package com.hg.redis;
  2.  
  3. import org.springframework.beans.factory.annotation.Autowired;
  4. import org.springframework.data.redis.core.StringRedisTemplate;
  5. import org.springframework.data.redis.core.ValueOperations;
  6. import org.springframework.stereotype.Repository;
  7.  
  8. import java.util.concurrent.TimeUnit;
  9.  
  10. /**
  11.  * @Description:
  12.  * @Author: jiangfan
  13.  * @CreateTime 2018/9/27 上午10:14
  14.  */
  15. @Repository
  16. public class RedisDao {
  17.     @Autowired
  18.     private StringRedisTemplate template;
  19.  
  20.     public void setKey(String key, String value) {
  21.         ValueOperations<String, String> ops = template.opsForValue();
  22.         ops.set(key, value);
  23.     }
  24.  
  25.     public String getValue(String key) {
  26.         ValueOperations<String, String> ops = this.template.opsForValue();
  27.         return ops.get(key);
  28.     }
  29. }
